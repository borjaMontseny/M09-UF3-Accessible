Spin Button

Un "spin button" és un element d'interfície gràfica d'usuari que s'utilitza comúment en aplicacions informàtiques per permetre als usuaris seleccionar un valor numèric dins d'un rang específic. Consisteix típicament en una caixa de text juntament amb botons de fletxa cap amunt i cap avall, que permeten augmentar o disminuir el valor mostrat a la caixa de text de manera incremental. Els usuaris poden fer clic a les fletxes o utilitzar el teclat per modificar el valor, la qual cosa resulta útil per seleccionar opcions numèriques de manera ràpida i precisa. Els "spin buttons" són comuns en interfícies de programari per configurar ajustos com l'hora, la data, la mida de la font, entre altres.

Com fer un Spin Button?

A continuació presentem com fer diferents de spin buttons

Originalment es fa amb un input type number, pero es podria utilitza també un un select

Input type number:

input type="number"

                        <label for="hores">Hores →</label>
                        <input type="number" id="hores" name="hores" value="1" min="0" max="23">

Select:

select

                        <label for="diesSetmana">Dies de la setmana →</label>
                        <select id="diesSetmana" name="diesSetmana">
                            <option value="dilluns">Dilluns</option>
                            <option value="dimarts">Dimarts</option>
                            <option value="dimecres">Dimecres</option>
                            <option value="dijous">Dijous</option>
                            <option value="divendres">Divendres</option>
                            <option value="dissabte">Dissabte</option>
                            <option value="diumenge">Diumenge</option>
                        </select>

Exemple

Input type number:

Escriu o escull un número:

Valor Mínim Valor Màxim Valor per defecte
0 23 1

Select:

Escull un dia de la setmana:
Valor per defecte
Dilluns

Accessibilitat amb el ratolí

Els usuaris que interactuen amb el ratolí han de ser capaços de fer clic als botons d'increment i decrement de l'Spin Button de forma fàcil i precisa.

Accessibilitat amb el teclat

Els usuaris que interactuen amb el teclat han de ser capaços de navegar fàcilment fins als botons d'increment i decrement del "Spin Button" utilitzant la tecla de tabulació (Tab) i la tecla de retrocés (Shift + Tab) per moure's cap endavant i cap enrere , respectivament, a través dels elements enfocables de la pàgina.
